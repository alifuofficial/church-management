# Church Management System

A modern, high-premium digital sanctuary designed to streamline church operations and enhance community engagement. This system provides a comprehensive suite of tools for managing members, sermons, events, and spiritual growth.

## ✨ Core Features

- **🕍 Digital Sanctuary**: A beautiful, responsive homepage with a creative hero section.
- **📖 Sermon Management**: Library of messages with video integration and scripture references.
- **📅 Event Scheduling**: Comprehensive calendar for weekly services and special gatherings.
- **🙏 Prayer Support**: 24/7 prayer request system and ministry support.
- **👥 Member Dashboard**: Personalized space for community members to connect.
- **⚙️ Admin Panel**: Robust control center for web presence, business logic, and maintenance.

## ✨ Technology Stack

Built with a modern, production-ready stack for performance and scalability:

### 🎯 Core Framework
- **⚡ Next.js 16** - React framework with App Router and Server Components.
- **📘 TypeScript 5** - Full type safety across the entire application.
- **🎨 Tailwind CSS 4** - Modern utility-first CSS styling.

### 🧩 UI & UX
- **🧩 shadcn/ui** - Accessible, high-quality component architecture.
- **🌈 Framer Motion** - Smooth micro-animations and transitions.
- **🎯 Lucide React** - High-premium icon system.
- **🎨 OKLCH Colors** - Perceptually uniform color palette for deep dark modes.

### 🗄️ Backend & State
- **🗄️ Prisma** - Type-safe ORM for database management.
- **🔐 NextAuth.js** - Secure authentication and role-based access control.
- **🐻 Zustand** - Lightweight and scalable state management.
- **🔄 TanStack Query** - Efficient data fetching and synchronization.

## 🚀 Getting Started

```bash
# Install dependencies
bun install

# Database setup
bun run db:push
bun run db:generate

# Start development
bun run dev
```
